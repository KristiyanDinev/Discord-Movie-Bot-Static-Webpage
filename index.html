This Privacy Policy explains how Movie Guess Bot collects, uses, and stores your data. We are committed to protecting your privacy and only collecting the minimum amount of data required for the Bot to function.

1. Information We Collect

To provide the guessing game functionality and maintain leaderboards, we collect and store the following data in a local SQLite database:

1.1 User Data

  • User ID: Your unique Discord User ID (Snowflake).
  • Username/Nickname: Your display name within the specific guild where the game is played.
  • Game Stats: Your number of correct guesses and current score.

1.2 Guild Data

  • Guild ID: The unique ID of the server where the Bot is installed.
  • Guild Name: The name of the server (used for the Global/Server Leaderboard).

2. How We Use Your Information

The collected data is used exclusively for the following purposes:

  • Game State Management: To track which user has guessed which movie and enforce the "One Attempt Only" rule.
  • Leaderboards: To display the "Member Leaderboard" (top players in a server) and the "Server Leaderboard" (guilds with the most correct guesses).

3. Data Storage and Security

All data is stored in a persistent SQLite database hosted on the machine running the Bot. We take reasonable measures to protect this data from unauthorized access. We do not sell, trade, or transfer your data to outside parties.

4. Data Retention

Data is retained indefinitely to maintain the history of the leaderboards. However, data may be cleared upon specific user request.
